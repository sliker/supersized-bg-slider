Supersized Jquery plugin integrated to Wordpress

== Description ==

This is the integration of the jQuery supersized backgroun plugin to wordpress, allowing to put background images on the wordpress site.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the whole plugin folder to your /wp-content/plugins/ folder.
1. Go to the Plugins page and activate the plugin.
1. Try your blog and check work properly.
